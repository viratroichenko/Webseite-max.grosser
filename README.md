
# Website für die Firma Max Großer - Heizung & Sanitär

## Beschreibung
Diese Webseite wurde für die Firma Max Großer im Bereich Heizung & Sanitär in Schöningen entwickelt.

Die Website dient dazu, die Dienstleistungen der Firma online zu präsentieren und potenziellen Kunden einen Überblick über Angebote und Kontaktmöglichkeiten zu geben.

## Funktionen
- Darstellung der Dienstleistungen im Bereich Heizung und Sanitär
- Übersichtliche und benutzerfreundliche Navigation
- Responsives Design für verschiedene Geräte (Desktop, Tablet, Smartphone)
- Kontaktbereich für Kundenanfragen

## Technologien
- HTML
- CSS
- JavaScript

## Ziel
Ziel des Projekts war es, eine moderne und funktionale Webseite für ein Handwerksunternehmen zu erstellen.
